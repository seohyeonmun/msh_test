README
- 문서현 :  
>  이번 프로젝트를 통해 KWS, STT, TTS를 하나의 파이프라인으로 연결하며  
>  음성 AI 서비스의 전체 흐름을 경험할 수 있었습니다.  
>  
>  특히 Whisper 기반 음성 인식 과정에서 정류소명이 정확히 인식되지 않는 문제를 해결하기 위해  
>  Levenshtein Ratio 기반 문자열 유사도 비교를 적용하며  
>  실제 사용자 환경을 고려한 예외 처리의 중요성을 배울 수 있었습니다.  
>  
>  그리고 협업 과정에서는 GitHub를 활용한 버전 관리와 역할 분담의 중요성을 경험했습니다.  
>  
>  다만 실제 다양한 소음 환경에서 충분한 테스트를 진행하지 못한 점은 아쉬움으로 남았습니다.  
>  향후에는 GPU 환경 최적화와 모바일 환경 지원,  
>  다양한 지역의 BIS API 연동 등을 통해 서비스를 확장해보고 싶습니다.  

- 김가람 :  
>  저는 본 프로젝트를 통해서 배운게 많은 거 같습니다.  
>  먼저 하나의 파이프라인으로 연결하는것을 통해 서비스의 워크플로우가 어떻게 이어지는가에 대한 통찰을 얻게 되었습니다.  
>  특히 오케스트레이터를 통해서 조율되는 워크플로우가 있음도 알게 되었습니다.    
>    
>  두번쨰로 tts를 맡으면서 추론과정속에서 아키텍처를 어떻게 최적화 할 수 있는지에 대한 고민을 한 것이 정말 중요한 경험이었다고 생각합니다.  
>  api를 통해서 모델 메모리 상주방식을 설계하고 응답지연문제를 획기적으로 줄인 경험은 훗날 크게 도움될거라 확신합니다.  
>    
>  마지막으로 혼자가 아닌 팀프로젝트로 팀원들과  GitHub를 통한 협업방식이 처음에는 낯설었지만 점차 시너지 효과가 있음을 알게되는 유익한 시간이였습니다.
